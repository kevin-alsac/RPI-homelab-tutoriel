# Mon Homelab Raspberry Pi 5

Vue globale du projet, architecture cible et etat d'avancement.

## Objectif

Construire un homelab fiable sur Raspberry Pi 5 avec acces distant securise, services conteneurises, HTTPS, Nextcloud et sauvegardes vers NAS.
